Wireframes — Locadora de Veículos

Trabalho Prático 2 · Front-end integrado à API do TP1

Média fidelidade, em tons de cinza, para guiar a construção das telas.

1. Painel (consultas ao banco)

2. Listagem + busca com filtros (2 campos)

3. Cadastro / edição (modal)

4. Carteira (recarga, saldo e pagamento)

5. Aluguéis + devolução

Telas previstas: Painel, Veículos, Aluguéis, Clientes, Carteira, Fabricantes e Categorias. Cada entidade tem cadastro, edição, exclusão e busca com filtros em pelo menos dois campos. A Carteira cobre as funcionalidades de recarga, consulta de saldo e pagamento.
